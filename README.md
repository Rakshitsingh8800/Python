# Concierge Multi-Agent System (Kaggle Project)

A Python-based **multi-agent system** designed to assist with daily tasks using AI-powered agents. Each agent specializes in a specific domain, integrating with **Google AI Studio** for intelligent content generation. All data is stored locally using **SQLite**, and the system provides a **command-line interface (CLI)** for easy interaction.
